## Time Tracking Features

- **Automatic Timers** - Start/stop timers for different tasks
- **Manual Entry** - Add time entries for past work
- **Hourly Rates** - Set different rates per client or project
- **Billable vs Non-billable** - Categorize time entries appropriately
- **Invoice Integration** - Automatically include tracked time in invoices
